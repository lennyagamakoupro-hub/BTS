## Comment l'ouvrir

1. **Double-clique** sur le fichier "LENNY-BTS-Transaction.html"
2. Il s'ouvre dans ton navigateur (Chrome, Edge, Safari, Firefox)
3. Tu n'as **AUCUN BESOIN d'internet** — tout est déjà dedans

## Si tu vois une "page Google" au lieu de l'app

Tu l'as ouvert dans un aperçu Gmail / Drive au lieu du navigateur. Solution :

- **Windows** : clic-droit sur le fichier → "Ouvrir avec" → Chrome / Edge / Firefox
- **Mac** : clic-droit → "Ouvrir avec" → Safari / Chrome
- **Téléphone** : ouvre l'app Fichiers, tape sur le fichier, choisis ton navigateur

Astuce : glisse le fichier directement dans un onglet de Chrome ouvert.

## Comment l'utiliser

- **Réviser** (gros bouton blanc sur le hero) → lance le diaporama du Module 11
- **Cliquer sur n'importe quelle carte** → ouvre la fiche détaillée avec Fiches / Mémos / Quiz / À propos
- **Bouton BINGE** (en haut à droite) → enchaîne tous les modules à la suite, façon marathon
- **Loupe 🔍** (en haut à droite, ou ⌘K / Ctrl+K) → recherche un module, un chapitre, une loi, un concept

### Raccourcis dans le diaporama
- ← → : naviguer entre les diapos
- Échap : quitter
- ⌘K / Ctrl+K : ouvrir la recherche
